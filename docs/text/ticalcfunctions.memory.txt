[Main]
Name=memory
Type=Enum
Header=calc_int.h
Definition=TicalcMemoryType memory

[Description]
This field allows to interpret the value returned in the memory parameter of the ticalcfunctions.directorylist function.

When set to MEMORY_FREE, free memory is returned.
When set to MEMORY_USED, used memory is returned.
