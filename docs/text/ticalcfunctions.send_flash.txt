[Main]
Name=send_flash
Type=Function
Header=calc_int.h
Definition=int (*send_flash) (const char *filename, int mode);

[Parameters]
filename : a file containing an FLASH application/OS/certificate
mode : a set of mask from @TicalcMaskMode@
Returns : 0 if error, an error code otherwise

[Summary]
Send the content of an FLASH file to the calculator.

[Description]
This function can send an FLASH application or an OS upgrade (AMS) to the calculator. Certificates are supported but has never been fully tested.

[See also]
@ticalcfunctions.recv_flash@
