[Main]
Name=ticalc_format_to_date
Type=Function
Header=calc_int.h
Definition=const char* ticalc_format_to_date (int value);

[Parameters]
value : a format specifier value between 1 and 8
Returns : the format specifier as a string

[Summary]
Make correspondance.

[Description]
The different format specifiers are:

<br>1:    "MM/DD/YY"
<br>2:    "DD/MM/YY"
<br>3:    "MM.DD.YY"
<br>4:    "DD.MM.YY"
<br>5:    "YY.MM.DD"
<br>6:    "MM-DD-YY"
<br>7:    "DD-MM-YY"
<br>8:    "YY-MM-DD"
